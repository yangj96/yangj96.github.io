1. 循环结束条件，忘记break，例如双指针循环条件

   ```
   while (j < n) 
   	if (a[i] == a[j])
   		j++;
   
   条件直接&&在while判断内，如果采用if形式循环内注意结束条件
   ```

3. for循环指针直接跳转

   ```
   for (int i = 0; i < n; i++) {
   	int j;
   	...
   	i = j;
   }
   for循环执行后i++，下一循环i实际上位j+1
   ```
